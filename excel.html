Work Schedule: This spreadsheet is used to create a work schedule for all employees of an organization. The spreadsheet calculates labor hours for each day and for the week. The spreadsheet includes charts to ensure coverage for 24 hour operation, and also to ensure proper management coverage.

Audit Results: This spreadsheet displays the results of an audit of all messages being sent out via Slate Deliver. The spreadsheet includes an overview tab for all messages, and is also organized, per tab, by communication stream. The results of each message are indicated by a color key.

Database: This spreadsheet is used as a database. The database includes English proficiency requirements for each program listed on our graduate program requirements webpages. The data is exported as XML, and is loaded into Cascade. This database increases efficiency, as bulk updates can be made to all webpages quickly.

Application Load: This macro-enabled spreadsheet is used to compare two files. One file from the CRM is inserted into the first tab of the workbook. The other audit file from the SIS is inserted into the second tab of the workbook. The macro is ran, and the results are of those applicants whose accounts are reviewed in Banner for matching.

Application Processing Time: This macro-enabled spreadsheet is used to calculate the average (mean, median, range, and mode) time it takes to admit applicants during any given period. CRM query results are inserted into the workbook tabs. Prior to this spreadsheet, this task would take approximately one week to complete.
